# 🎓 StudyForge - AI Study Planner

## 📖 About
StudyForge is a complete, all-in-one study management platform that helps students plan, track, and execute their exam preparation efficiently. Powered by AI, it creates personalized study schedules based on your exam date, subjects, and difficulty preferences.

## ✨ Key Features

### 🤖 AI Schedule Generator
- Enter your exam date and subjects
- Set difficulty levels (Easy/Medium/Hard)
- AI creates balanced day-wise schedule
- Automatic revision days (spaced repetition)

### ⏳ Pomodoro Timer
- 25-minute focus sessions
- 5-minute short breaks
- 15-minute long breaks
- Track daily focus sessions

### 📊 Progress Tracking
- Daily task checklist
- Visual progress bar
- Subject-wise completion stats
- Daily streak counter
- XP and level system

### 🔐 User System
- Sign up / Login with email
- All data saved locally
- Pick up where you left off

### 📱 Responsive Design
- Works on mobile, tablet, and desktop
- Dark theme optimized
- Smooth animations

## 🚀 How to Use

1. **Sign Up** - Create a free account
2. **Enter Exam Details** - Exam name, date, daily study hours
3. **Add Subjects** - Name and difficulty level
4. **Generate Plan** - AI creates your personalized schedule
5. **Use Timer** - Pomodoro timer for focused study
6. **Track Progress** - Check off daily tasks
