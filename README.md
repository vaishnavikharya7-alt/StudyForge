# 📚 AI Study Planner 🤖

An intelligent web application that helps students plan their studies efficiently using AI-based scheduling and smart tracking.

---

## 🚀 Problem Statement

Many students struggle with:
- ❌ Not knowing what to study next  
- ❌ Wasting time creating timetables  
- ❌ Poor revision planning  
- ❌ Lack of consistency and tracking  

As exams approach, this leads to stress, confusion, and inefficient preparation.

---

## 💡 Solution

The **AI Study Planner** solves this by:
- Generating a **personalized study schedule** based on exam date, subjects, and daily study hours  
- Automatically distributing subjects for balanced learning  
- Including **revision cycles** for better retention  
- Helping students stay consistent with **progress tracking**

---

## ✨ Features

- 🧠 AI-generated study plans  
- 📅 Daily task management  
- 📊 Progress tracking dashboard  
- 🔁 Smart revision system (based on spaced repetition)  
- 🔄 Auto-rescheduling for missed tasks  
- 🔐 User authentication (login/signup)  
- 🎯 Clean and distraction-free UI  

---

## 🛠️ Tech Stack

- Frontend: React / Next.js  
- Backend: Firebase  
- Authentication: Firebase Auth  
- Database: Firestore  
- AI Logic: Custom scheduling algorithm  

---

## ⚙️ How It Works

1. User enters:
   - Exam date  
   - Subjects  
   - Daily study hours  

2. System calculates:
   - Total days available  
   - Subject priority  

3. AI generates:
   - Day-wise study plan  
   - Balanced schedule  
   - Revision slots  

4. User:
   - Marks tasks complete  
   - Tracks progress  
   - Gets updated plans
